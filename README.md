# 复制记录板浏览器插件

一个可以记录所有复制内容的浏览器插件，支持在网页上选取文字复制并保存到本地历史记录中。

## 功能特点

- 📋 在网页上选取文字时显示复制按钮
- 🗂️ 自动保存所有复制内容到本地存储
- 🔍 支持搜索历史记录
- ⭐ 可以收藏重要内容
- ⚙️ 可配置保存文件夹和最大记录数量

## 安装方法

1. 下载或克隆本项目
2. 打开Chrome浏览器，进入 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"，选择本项目文件夹

## 使用方法

1. 在网页上选取文字，会出现复制按钮
2. 点击按钮复制内容并自动保存到历史记录
3. 点击浏览器工具栏中的插件图标，可以查看和管理历史记录
4. 在设置页面可以配置保存位置和其他选项

## 跨软件复制检测

要检测从其他软件（如微信、Word等）复制的文本，需要额外安装一个本地应用。这部分功能将在未来版本中提供。
