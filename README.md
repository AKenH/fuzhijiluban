# 复制记录板浏览器插件

一个简洁实用的浏览器插件，可以自动记录所有文本复制内容。

## 功能特点

- 📋 在网页上选取文字时显示优雅的复制按钮
- 📝 自动记录所有文本复制内容
-  强大的搜索功能
- ⭐ 可以收藏重要内容
- ⚙️ 可配置保存位置和最大记录数量
- 🎨 精美的现代化UI设计

## 安装方法

1. 下载或克隆本项目
2. 打开Chrome浏览器，进入 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"，选择本项目文件夹

## 使用方法

### 方式一：悬浮复制按钮
1. 在网页上选取文字，会出现复制按钮
2. 点击按钮复制内容并自动保存到历史记录

### 方式二：键盘快捷键
1. 选中文字后按 `Ctrl+C`（或 `Cmd+C`）
2. 文本会自动复制并保存到历史记录

### 查看记录
- 点击浏览器工具栏中的插件图标，可以查看和管理历史记录
- 在设置页面可以配置保存位置和其他选项
- 使用搜索框可快速查找历史记录
